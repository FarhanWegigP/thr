# THR Tio 🌙

## Setup

```bash
npm install
npm run dev
```

## Yang perlu kamu isi di folder `public/`

```
public/
├── memes/
│   ├── lebaran1.jpg   ← marquee meme lebaran (5 file)
│   ├── lebaran2.jpg
│   ├── lebaran3.jpg
│   ├── lebaran4.jpg
│   ├── lebaran5.jpg
│   ├── thr1.jpg       ← marquee meme minta THR (4 file)
│   ├── thr2.jpg
│   ├── thr3.jpg
│   └── thr4.jpg
└── qris.png           ← screenshot QRIS Gopay kamu
```

Format bebas: jpg, png, gif, webp semua oke.

## Deploy ke Vercel

```bash
npx vercel --prod
```

atau connect repo GitHub ke vercel.com langsung dari dashboard.
